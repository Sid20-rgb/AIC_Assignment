# ITS- ShapeSmart

## Steps/Instructions to run the system

### To get and run the code from GitHub:
* Inside the repository 'AIC_Assignment', clone this repo
* Step 1: open terminal in your device and type git clone https://github.com/Sid20-rgb/AIC_Assignment.git
* Step 2: Right click AIC_Assignment folder (from the path where you have just cloned)
* Step 3: Open in AIC_Assignment folder in terminal
* Step 4: Run the command python main.py
(If it asks for to install any module that's not in your device)
* Step 6: pip install <that-missing-module>
* Step 7: python main.py


### To run the code from the provided 250188231_PythonUICode.zip
* Step 1: Download the 250188231_PythonUICode.zip
* Step 2: Extract/Unzip the file
* Step 3: Open 250188231_PythonUICode where you can find ShapeSmart-ITS-Code folder
* Step 4: Right click ShapeSmart-ITS-Code folder and Open in terminal
* Step 5: Type command code . in terminal
* Step 6: After the command opens the folder in VS code or any other IDE, simply just run python main.py in the VS Code terminal.
(If it asks for to install any module that's not in your device)
* Step 6: pip install <that-missing-module>
* Step 7: python main.py 

### To run Ontology Integration test:

* Step 1: In the terminal, in the root directory where the main.py is, run cd ontology
* Step 2: Inside ontology folder, there is ontology_test.py 
* Step 3: Run python ontology_test.py

### Some important modules to install beforehand:
* pip install customtkinter
* pip install rdflib
* pip install pillow
* pip install customtkinter rdflib pillow

### In short, it is to Run:
1. pip install customtkinter
2. python main.py

If you still get ModuleNotFoundError, make sure you run from the project root (folder containing main.py).
